# Pinhead map icon font

This package contains the [Pinhead map icons](https://pinhead.ink) as a web font.

## Usage

1. Copy `pinhead.ttf` and `pinhead.css` to your website's directory.
2. Link the CSS file in your `<head>` element: `<link href="pinhead.css" rel="stylesheet"/>`
3. Add icon elements with a class in the format `pinhead-ICON_ID`, like so: `<span class="pinhead-box_truck"></span>`
4. Style like text, for example: `font-size: 15px; color: navy; text-decoration: underline solid 2px red;` 

See the included `/preview.html` page as an example.
